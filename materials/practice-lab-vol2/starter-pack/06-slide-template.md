# Monthly Briefing — SLIDE TEMPLATE (8 slides)

> Markdown spec for the visual template. To deploy: build the matching 8-slide Google Slides deck
> following the visual cues below, then export as .pptx.
>
> Visual identity: clean light background (#FFFFFF or near-white), one accent color (deep teal or
> slate blue recommended), plain sans-serif typeface (e.g., Inter or Source Sans), no decorative
> gradients. Charts use the accent color only — no multi-color palettes unless a legend is required.
> This spec is the primary visual reference for Stage 3b design work.
